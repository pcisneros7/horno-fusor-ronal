# horno-fusor-ronal
Modelado y control térmico de un horno fusor de aluminio . Compara un modelo clásico (función de transferencia con retardo) frente a un modelo LSTM basado en datos, considerando perturbaciones reales del proceso. Busca identificar los parámetros del sistema (K, τ, L) y determinar qué enfoque describe mejor la dinámica real del horno.
